What is a Histogram?

A Histogram divides all the given values into a certain number of "bins" and shows the vertical height of each bin as the number of values in that bin

What is a Bar Graph?

A Bar Graph takes each value and shows its value as a certain height

What is a Scatter Plot?

A Scatter Plot takes two sets of related values and plots them. This shows the relationship between the values
